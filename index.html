System Architecture Overview

This application operates on a Forward-Chaining Inference Engine, meaning it starts with the user's data (the preferences) and works forward to a conclusion (the recommendation).

  • Backend Logic: Python (Flask) hosts the core **knowledge base**â€”a collection of explicit rules (`rules` list) that define product criteria.
  • Inference Mechanism: The `recommend()` function checks if a rule's conditions are a **subset** of the user's three input parameters.
  • Robustness: The system is designed with multiple **fallback rules** to ensure a meaningful recommendation is provided, even if highly specific criteria are not met.

The modern frontend is built with HTML, JavaScript, and styled using Tailwind CSS, ensuring a fast, responsive, and aesthetically pleasing user experience that reflects the project's AI theme.
